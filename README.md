## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) 18+

### Installation

```bash
git clone https://github.com/<your-username>/devstash.git
cd devstash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```
